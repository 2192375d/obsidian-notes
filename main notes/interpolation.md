# why interpolate?
- plot a smooth curve through discrete data points
- quick and easy evaluation of a mathematical function
- replacing a "difficult" function with an "easy" one
- differentating or integrating tabular data

# interpolation (simplest definition)
For data $(t_{i},y_{i}),i=1,\dots,n$,
It's __interpolating function__, or simply, __interpolant__ is a function $f$ S.T.
$$
f(t_{i})=y_{i},i=1,\dots,n
$$
