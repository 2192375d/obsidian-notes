# theorem
Given $\displaystyle\sum_{n=M}^\infty ar^n$, where $a,r\in\mathbb{R},a\neq 0$
- if $|r|<1$, then $\sum ar^n$ converges to $\displaystyle\frac{ar^M}{1-r}$
- if $|r|\geq1$, then $\sum ar^n$ diverges
