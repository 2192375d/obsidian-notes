# theorem
Given $\displaystyle\sum_{n=0}^\infty a_{n}$, consider $f(x)$ that satisfies
- continuous
- decreasing
- positive
- $a_{n}=f(n),\forall n\in\mathbb{N}$
on $[0,\infty]$, then 
$$
\displaystyle\sum_{n=1}^\infty a_{n}\text{ converges } \leftrightarrow\int_{1}^\infty f(x)dx\text{ converges }
$$
