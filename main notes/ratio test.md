# theorem
Given $\sum a_{n},a_{n}\neq 0$, define $L=\lim_{ n \to \infty } |\frac{a_{n+1}}{a_{n}}|$
- if $L<1$, then $\sum a_{n}$ absolutely converges
- if $L>1$, then $\sum a_{n}$ diverges
