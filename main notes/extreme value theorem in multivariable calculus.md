# extreme value theorem (EVT)
Suppose $D\subset\mathbb{R}^n$ is closed and bounded
If $f:D\subset \mathbb{R}^n\to\mathbb{R}$ is continuous, then there are points $x_{0}$ and $x_{1}$ in $D$ where $f$ attains its global maximum and minimum values.
