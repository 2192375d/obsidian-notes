# main value theorem for double integral
Let $f:D\to\mathbb{R}$ be continuous on $D$, where $D$ is an elementary region, then there is some point $(x_{0},y_{0})$ S.T.
$$
\iint_{D}f(x,y)dA=f(x_{0},y_{0})\text{Area(D)}
$$​​
