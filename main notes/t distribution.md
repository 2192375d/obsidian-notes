# t distribution
If $Z\sim N(0,1)$ and $V\sim \chi^2(n)$, while $Z\perp V$, then $\displaystyle T=\frac{Z}{\sqrt{ \frac{V}{n} }}$ follows a __t distribution__, with parameter $n$
