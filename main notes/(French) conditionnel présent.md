# conditionnel présent
Le conditionnel présent est formé du radical du __futur__ et de l'__indicatif__ de présent

_example_
- __Si je peux__, j'irai sur Internet
- __Si je pouvais__, j'achèterais un ordinateur

	Avec __si je peux__ pour les expressions qui expriment le futur
Avec __si je pouvais__ pour les expressions qui expriment une situation hypothétique

# 
Si + sujet + présent -> future simple
Si + sujet + imparfait -> conditionnel présent
