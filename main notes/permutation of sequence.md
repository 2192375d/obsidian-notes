# permution of sequence
Let $\sigma:I\to A$ and $\tau:I\to A$ be finite sequences over $A$, then $\tau$ is a __permutation__ of $\sigma$ if there exists a bijective function $f:I\to I$ S.T.
$$
\forall i\in I,\tau(i)=\sigma(f(i))
$$
