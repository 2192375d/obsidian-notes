# definition (x-simple region)
A region $R \subset \mathbb{R}^2$ is $y$-simple if there exist continuous funcitons $\varphi_{1},\varphi_{2}:[a,b]\to \mathbb{R}$ S.T.
$$
R=\{ (x,y):a\leq x\leq b \text{ and } \varphi_{1}(x)\leq y\leq\varphi_{2}(x) \}
$$
# definition (y-simple region)
A region $R \subset \mathbb{R}^2$ is $x$-simple if there exist continuous funcitons $\psi_{1},\psi_{2}:[c,d]\to \mathbb{R}$ S.T.
$$
R=\{ (x,y):c\leq y\leq d \text{ and } \psi_{1}(y)\leq x\leq\psi_{2}(y) \}
$$
# definition (simple)
A region $R\subset\mathbb{R}^2$ is __simple__ if it is $x$-simple and $y$-simple.
We also call it an __elementary__ region
