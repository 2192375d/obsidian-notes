# triads
- major: root, root + M3, root + M3 + m3
- minor: root, root + m3, root + m3 + M3
- augmented: root, root + M3, root + 2M3
- diminished: root, root + m3, root + 2m3

## chord inversion
Consider a base of C major
Then root is: CEG, with bass note C
- first inversion: EGC, and bass note is E
- second inversion: GCE, and bass note is G
