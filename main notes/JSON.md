# JSON
Fully name __Java Script Object Notation__
- supported by many programming languages
- behavior is language independent
- uses readable text to represent data in the form of key/values

example:
```JSON
{
	"name": "Alex",
	"age": 25,
	"address": {
		"country": "Canada",
		"city": "Toronto"
	}
}
```
