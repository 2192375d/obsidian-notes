# order statistics
For RVs $X_{1},\dots,X_{n}$, the kth order statistic ($X_{(k)}$) is the kth smallest variable:
- $X_{1}=min(X_{1},\dots,X_{n})$
- $X_{n}=max(X_{1},\dots,X_{n})$
- $X_{(1)}\leq X_{(2)}\leq\dots\leq X_{(n-1)}\leq X_{(n)}$
