# tempo

| name        | meaning                 | BPM     |
| ----------- | ----------------------- | ------- |
| grave       | very very slow (solemn) | 20-40   |
| lento/largo | very slow               | 40-60   |
| adagio      | slow                    | 60-80   |
| andante     | walking pace            | 80-110  |
| moderato    | moderately              | 110-130 |
| allegro     | fast                    | 130-170 |
| presto      | very fast               | 170-200 |
| pretissimo  | very very fast          | 200+    |
