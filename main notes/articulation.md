# legato 
![[Pasted image 20250917153431.png|left]]
Connect each note smoothly to the next one, a slur may be used to indicate legato articulation.

# accent
![[Pasted image 20250917153638.png|left]]
Emphasize the beginnings of each note, hold each note for almost its hold duration, with a slight decrease in volume across the note.

# staccato
![[Pasted image 20250917153707.png|left]]
Notes are played short, with a seperation between them

# marcato
![[Pasted image 20250917154500.png|left]]
The beginning of the note is accented and notes are played quite short
