# independent and identically distributed RVs (i.i.d.)
A set $X_{1},\dots,X_{n}$ of random variables is __i.i.d.__ if the variables are indepdent and all $n$ variables have the same distribution.

$X_{1},\dots,X_{n}$ is also refered as a sample from the common distribution
