# matrix
Usually denoted as $A=[a_{i,j}]$ where $i$ is the row and $j$ is the column
By $A$ is a $N \times M$ matrix, we mean $A$ has $N$ rows and $M$ columns

## matrix representations
The following 3 represent each other:
- an $N \times M$ matrix
- a linear system of $M$ variables and $N$ equations
- a set of $M$ vectors with $N$ components
