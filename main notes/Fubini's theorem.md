# theorem
Let $f$ be integrable function with rectangular domain $R=[a,b]\times[c,d]$, then
$$
\int_{a}^b\int_{c}^df(x,y)dydx=\int_{c}^d\int_{a}^bf(x,y)dxdy=\iint_{R}f(x,y)dA
$$
Note that this theorem also holds for any domain $R$, doesn't have to be rectangular domain
