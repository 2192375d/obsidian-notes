# selection sort
Selection sort is done through repeatedly finding the smallest element and swap it with the earlier elements.
- set $i = 0$
- find smallest element from entry $i$ to $n$
- swap it with $i$
- $i = i+1$

## time complexity
time complexity: $O(N^2)$ (same for worse/best/average case)
