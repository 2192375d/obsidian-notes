# solvability of a matrix
For a $n\times n$ matrix $A$, 
- If $A$ is nonsingular, then the system $Ax=b$ always has a unique solution $x=A^{-1}b$ regardless of the value for $b$
- Otherwise, $A$ either has no solution or infinite solution (parallel)
