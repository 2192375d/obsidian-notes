# scale degree
Consider W as whole note, H as half note, then
- tonic: root
- supertonic: root + W
- mediant: root + 2W
- subdominant: root + 2W + H
- dominant: root + 3W + H
- submediant: root + 4W + H
- leading tone: root + 5W + H
- tonic:  root + 6W
