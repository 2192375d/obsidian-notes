# powerset from B36 tb(0.1.3)
Let $A$ be a set, then the **powerset** of $A$ is the set of all subsets of $A$, denoted $\mathcal{P}(A)$

(Note that $|\mathcal{P}(A)|=2^{|A|}$)
(Note that $\emptyset \in P(A)$ for any set A)
