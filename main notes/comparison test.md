# theorem
Given $\sum a_{n},\sum b_{n}\sum c_{n},$
- if $0\leq a_{n}\leq b_{n}$ and $\sum b_{n}$ converges, then $\sum a_{n}$ converges
- if $0\leq c_{n}\leq b_{n}$ and $\sum c_{n}$ diverges, then $\sum b_{n}$ diverges
