# midterm 2:
Pour les statements en conditionel,
- imparfait + passé composé -> hypothèse irrélle dans le présent ou futur
- plus-que-parfait + conditionel passé -> hypothèse irréelle dans le passé

conjonctions suivies du subjonctif
- à condition que: _provided that_
- à moins que: _unless_
- afin que: _so that_
- avant que: _before_
- bien que: _although_
- de peur/crainte que: _for fear that_
- de sorte que: _so that_
- en attendant que: _while, until_
- jusqu'à ce que: _until_
- malgré que: _in spite of_
- pour que: _in order that_
- pourvu que: _provided that_
- quoique: _although_
- sans que: _without_
