## meaningful names
Names must be:
- intention-revealing
- able to make meaningful distinction
- pronounceable
- searchable

class names must have noun or noun phrase name
methods must have verb or verb phrase name
one word per concept
avoid using one word for two purposes

## comments
If a code is clear and expressive, then it does not need comment
Here are the situations when comments are useful:
- legal comment
- explanation of intent
- warning of consequences
