# matrix condition properties
1. For any matrix $A$, $cond(A)\geq1$
2. $cond(I)=1$
3. For any permutation matrix $P$, $cond(P)=1$
4. For any matrix $A$ and nonzero scalar $\gamma$, $cond(\gamma A)=cond(A)$
5. For any diagonal matrix $D=diag(d_{i})$, $\displaystyle cond(D)=\left( \frac{\max|d_{i}|}{\min|d_{i}|} \right)$
