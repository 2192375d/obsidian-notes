# vector space
Let $V$ be a vector space over $\mathbb{F}$, 
Let $u,v,w\in V$,
Let $c,d\in\mathbb{F}$
Then, $v+w,rv\in V$ and
1. $u+v=v+u$
2. $u+(v+w)=(u+v)+w$
3. $\exists 0\in V$ S.T. $u+0=u$
4. $\exists u'\in V$ S.T. $u+u'=0$
5. $c(u+v)=cu+cv$
6. $(c+d)v=cv+dv$
7. $cdv=c(dv)$
8. $\exists 1\in\mathbb{F}$ S.T. $1v=v$
